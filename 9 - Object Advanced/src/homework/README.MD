3. Создать фабричную функцию для генерации объекта Архив
   1. с закртым для прямого чтения/записи хранилищем.
   2. с реализацией следующих методов:
      1. add - добавляет запись в архив.
      2. delete - удаляет запись из архива, возвращает удаленную запись.
      3. find - ищет запись в архиве, возвращает запись.
      4. filter - фильтрует записи в архиве, возражает фильтрованные записи.
      5. each - выполняет некую функцию для каждой записи в архиве.
      6. take - возвращает срез архива по индексам from, to.
      7. isEmpty - возвращает логическое значение, является ли архив пустым.
      8. count - возвращает количество записей в архиве.
4. Создать фабричную функцию для генерации объекта Пользователь
   1. со следующим набором полей:
      1. firstName - имя пользователя.
      2. lastName - фамилия пользователя.
      3. year - год рождения пользователя.
      4. month - месяц рождения пользователя.
      5. day - день рождения пользователя.
   2. с реализацией следующих геттеров:
      1. fullName - возвращает конкатенацию имени пользователя и фамилии.
      2. age - возвращает возраст пользователя в виде целого числа, относительно текущего года, месяца и дня.
